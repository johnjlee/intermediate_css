CSS annoyance #1: width doesn't always set the width.

Did you notice that the "100 pixel box" was actually 102 pixels wide? (Go back and check)


Setting the width property sets the width of the content, not the whole container!

What we know so far about width

  • You can set the width of a block element, but it still eats the parent's full width.
  • You can't set the width of an inline element; it's totally meaningless.
